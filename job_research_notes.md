## üí° Tips for Student Job Seekers

### Best Time to Apply
- **Summer internships:** Apply October-February for summer positions
- **Co-op terms:** Check your university's co-op portal for postings
- **Part-time:** Available year-round

### What Employers Look For in Students
1. **Personal projects** - GitHub portfolio, hackathon projects
2. **Coursework** - Relevant CS/Engineering courses completed
3. **Enthusiasm** - Willingness to learn and contribute
4. **Basic skills** - Programming fundamentals, version control

### Recommended Skills to Develop
- **Programming:** Python, JavaScript, Java (pick 1-2 to focus on)
- **Web basics:** HTML, CSS, React or Vue
- **Tools:** Git, VS Code, basic command line
- **Soft skills:** Communication, teamwork, time management
